# fc0 Launcher

A launcher for @valoeghese's game called *2fc0f18*.

## Features

- Launching the game
- Downloading the libraries automatically
- Loading mods

## Credits

- [Jankson](https://github.com/falkreon/Jankson) by Falkreon
